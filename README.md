#  Airline Delay & Performance Analytics

An end-to-end Data Analytics project that analyzes over **5 million commercial airline records** to identify delay patterns, evaluate airline performance, and generate actionable business insights using **Python, SQL, and Power BI**.

---

## Project Overview

Flight delays affect operational efficiency, increase airline costs, and reduce passenger satisfaction. This project demonstrates a complete analytics workflow—from data cleaning to interactive dashboard development—to uncover the key drivers of flight delays and support data-driven decision-making.

The project includes:

* Data Cleaning with Python
* Data Analysis using SQL
* Interactive Dashboard in Power BI
* Business Insights and Recommendations
* Comprehensive Project Report

---

##  Objectives

* Clean and prepare raw airline data for analysis.
* Explore flight delay trends using SQL.
* Build an interactive Power BI dashboard.
* Analyze airline performance across routes, carriers, and time.
* Generate actionable operational recommendations.

##  Key Business Insights

### Late Aircraft Delays

Late-arriving aircraft are the largest contributor to overall delays, creating cascading disruptions throughout the day's schedule.

### Sunday Experiences the Highest Delays

Sunday records the highest average delay, indicating increased operational pressure during weekend return travel.

### High-Traffic Airport Hubs

Major hubs such as Atlanta (ATL) and Dallas/Fort Worth (DFW) experience the highest traffic volumes, making them critical locations for operational improvements.

### Delay Patterns Increase Throughout the Day

Morning departures are generally more punctual, while delays accumulate during afternoon and evening operations.
